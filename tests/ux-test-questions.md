# UX Test Questions

A reference set of questions for manual UX testing of the Question Coach app.
Use these as sample input during Stage 2 (Produce Questions) to exercise the
full QFT workflow — classification, prioritisation, and discussion.

Questions are categorized as **[O]** (open) or **[C]** (closed) to help verify that
the Stage 3 categorisation UI correctly handles both types.
